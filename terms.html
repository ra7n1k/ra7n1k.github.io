ゲーム

これはMinecraftサーバーに適用されます。

  1. このサーバーには経済システムプラグイン (JeconおよびVault) を導入しており、通貨単位はやきとりサーバードル / YSDとします。[^3] [^4]
  2. サービスへの負荷が高い場合、一定時間 (場合により異なる) 以上操作を行っていないユーザーはキックする場合があります。
  3. これらに反した場合はゲーム内でBANやキックなどの対応をモデレーター (マネジャー/モデレーター) の判断に基づいて行うことができます。

アイテム

  1. 公共の農場や植林場、牧場で収穫やと殺を行ったときは必ず同数以上、繁殖させるか種や苗を植えなくてはなりません。
  2. 資源の採取採掘は資源ワールド (ワールド名: resources) で行ってください (生活ワールドでは行わないこと)。
  3. すべてのプレイヤーがアクセスできるワールドでスポーン周辺においては可能な限りクライアントに負荷をかけるストラクチャー[^5]を作成しないでください。
  4. X-Ray[^6]や一括破壊系Mod[^7]の使用を許可しますが、Admin Shop[^8]以外での鉱石やいくつかのアイテム (取引禁止アイテム) の売買を禁止します。
  5. 取引はアイテム基本価格から上下5YSDの取引まで許可します。

土地と構造物

  1. ネザー (ワールド名: world_nether) は資源を採集することのみを想定しています。マネージャーが特別に指示した場合を除き、建築を保護する義務はありません。
  2. モデレーターはあらゆる範囲に立ち入る権限があり、規範に違反する建築などがあった場合通告後撤去などを行うことができます。
  3. 建物を建てる際は地面が道路と1マス、他の建物と3マス以上の空きがある場所で建設してください。
  4. 1つの建物に利用できる土地範囲の目安はスポーン地点からの距離によって決定します。

鉄道と道路

  1. 自分の敷地外に鉄道路線の建設 (延伸含む) を行いたい場合は、Discordサーバーのsupport/infrastructural-supportチャンネルに路線の幅から約1チャンク (32×32ブロック) 先までの沿線住民の7割以上 (地下鉄の場合は半分以上) の賛成データと路線計画を提出し、その通りに建設 / 稼働させてください。
  2. 私用・私有を禁止するので、必ず公共路線として使えるように設計し、稼働させるてください。
  3. ストラクチャーはモデレーターが管理できます。

関連するすべてのストラクチャーはこの利用規約で定められたあらゆる理由に基づき、設計が変更たり、一度承認された計画の許可が取り消される可能性があります。

モンスタートラップ

  1. モンスタートラップを作成する場合は必ず公共施設として利用できるように設計し、稼働させてください。また、稼働/不稼働を切り替えられるように設計してください。
  2. 建築 / 改築を望む場合は必ずDiscordサーバーのsupport/monster-trapチャンネルに計画を提出し、その通りに建設 / 稼働させてください。
  3. 必要としているものと同じ種類の公共モンスタートラップがある場合は、可能な限り新たに建築するのではなく既存のものを利用してください。
  4. ストラクチャーはモデレーターが管理できます。

関連するすべてのストラクチャーは利用規約で定められたあらゆる理由に基づき、設計が変更たり、一度承認された計画の許可が取り消される可能性があります。
