やきとりサーバー


Minecraft: Java Editionのサバイバル生活サーバーです。

概要

このサーバーは無料のJava Editionサーバーです。経済制度が導入されたサバイバルの生活 (採掘や農業、交易、建築、コミュニケーションなど) やクリエイティブによる無制限の建築を楽しむことができます。

以下の規約に同意していただければ誰でもアクセスすることができます。

規約
  1. チートModや不正アカウントの使用を禁止
  2. X-Rayなどのリソースパックの利用を許可するが、Admin Shopを除いて鉱石の売買を禁止する
  3. 他の人に不快感を与える行動、言動を禁止
  4. サーバーに多大な負荷を掛けることを禁止
  5. 広告活動 (他サーバーなど含む) などのサービス提供を妨害する行為を禁止
  6. 増殖バグを利用したアイテム生産を禁止
  7. 運営はゲーム内情報の編集、複写と削除権限を所持する
  8. 利用に伴う損害および事故に関して、一切の責任を負わない
