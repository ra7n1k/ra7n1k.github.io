やきとり サーバー


Minecraft: Java Editionのサバイバル生活サーバーです。

概要

このサーバーは無料のJava Editionサーバーです。経済制度が導入されたサバイバルの生活 (採掘や農業、交易、建築、コミュニケーションなど) やクリエイティブによる無制限の建築を楽しむことができます。

次の利用規約に同意していただければ誰でもアクセスすることができます。

サービス利用規約 (要約と抜粋)
  1. 日本のインターネットに関する法律とあなたの居住国の法律、MinecraftとDiscordの規約と利用規約を遵守すること
  2. あらゆる不正を禁止
  3. X-Rayや一括破壊系Modの使用を許可するが、モデレーターの指定したショップ以外での鉱石といくつかのアイテムの売買を禁止する
  4. 暴言の他、強要や脅迫などを禁止
  5. あらゆるアイデンティティに基づく攻撃を禁止
  6. サービスに負荷を掛けることを禁止
  7. プロジェクトに関連しない宣伝を禁止
  8. 利用に伴うあらゆる損害関して私たちは一切責任を負わない

参照してください: サービス利用規約 (全て)
