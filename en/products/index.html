DuckDuckGo+

This Firefox add-on is an extension for the privacy-friendly search engine DuckDuckGo .

Features

  • Remove promotions about DuckDuckGo
  • Change the link color that is difficult to see in dark mode

Features

  • Very lightweight
  • Do not collect user information
  • More features to come

Caution

  • This add-on was created for the desktop version of Firefox and is not guaranteed to work on mobile versions of Firefox or Chromium
  • This add-on is available on Chrome Web Store and Not distributed by AMO
  • For this add-on to work, go to about:addons from DuckDuckGo+ > Permissions > *://duckduckgo.com Access to stored datamust be enabled
  • Automatic updates are not currently enabled for this add-on. A new version with automatic updates enabled will be distributed on this site and AMO in the near future

Frequently asked questions

Enabled access to stored data on *://duckduckgo.com even though does not collect user information Why should I?

This addon applies new styles. In CSS, you have to specify a selector and get the identifier of an element in the page (for example, the title of a search result). However, this action is completed on the client and no information is sent to the server.

Why not distribute it on Chrome Web Store or AMO?

The reason why we do not distribute extensions on the Chrome Web Store is because the initial registration is paid. I don't have that much luxury. Registration on AMO is free, but I was referring to a somewhat outdated document and realized that Manifest version 3 extensions should not be published on AMO. The next version will also be distributed on AMO.
