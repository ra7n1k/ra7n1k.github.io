New Registration

Do you want to start a new company? Information is here.

Information

To establish a company that operates in the game, you will need to register the following information in our database.

  • Company name
  • Representative Minecraft player name
  • Representative's Discord user ID
  • Business details
  • Create a homepage (Y/N)

Home page

If you want to create a homepage for your company, please follow the following requirements

All platforms

  • All pages include representative contact information and a link to the Yakitori server
  • Desktop and mobile compatible
  • Does not contain libel or slander

https://www.yakitori.tk/companies (can be uploaded to this web server)

n
  • A page is one to four HTML files
  • Less than 5 CSS and Java Script files
  • Less than 10 images (If you would like to prepare multiple versions of <picture>, please let us know)
  • Folder and file names are half-width alphanumeric characters
  • Do not use server languages such as PHP
