Land and structures

  1. The Nether (world name: world_nether) is intended only for gathering resources. There is no obligation to protect the building unless specifically directed by the Manager.
  2. Moderators have the authority to enter any area, and if there is a building that violates the standards, they can remove it after notifying them.
  3. When constructing a building, please construct it in a place where the ground has 1 space between the road and at least 3 spaces between other buildings.
  4. The approximate land area available for one building is determined by the distance from the spawn point.

Railways and roads

  1. If you would like to construct (including extensions to) a railway line outside your premises, please contact support/ on the Discord server. Submit the approval data and route plan of more than 70% (more than half in the case of subways) of residents along the line from the line width to about 1 chunk (32 x 32 blocks) to the infrastructure-supportchannel, and Build/operate on the street.
  2. Private use or ownership is prohibited, so please be sure to design and operate the route so that it can be used as a public route.
  3. Structures can be managed by moderators.

All associated structures are subject to change in design and withdrawal of planning permission once approved for any reason set out in these Terms and Conditions.

Monster traps

  1. If you create a monster trap, please be sure to design and operate it so that it can be used as a public facility. Also, please design it so that it can be switched between operation/non-operation.
  2. If you want to build/renovate, be sure to post your plans in the support/monster-trap channel on the Discord server. Submit it and build/operate accordingly.
  3. If you have a public monster trap of the same type as what you need, use the existing one instead of building a new one whenever possible.
  4. Structures can be managed by moderators.

All associated structures are subject to change in design and withdrawal of planning permission once approved for any reason set out in the Terms and Conditions.
